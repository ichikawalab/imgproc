## 7.2 値画像処理  
### 純粋なモルフォロジ処理（演習7-3考察）

このリポジトリは、医用画像処理工学演習の一環として **モルフォロジ処理（Erosion / Dilation）** を実装・考察するための教材です。  
対象画像は TIFF/PNG を想定し、構造要素は十字型を使用しています。

---

## 📌 内容
- 原画像の読み込み（OpenCV）
- 十字型構造要素による **Erosion（収縮）**
- 十字型構造要素による **Dilation（膨張）**
- 結果の可視化（Matplotlib）
